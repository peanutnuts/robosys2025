# xspell: スペルチェックを高精度でするコマンド

標準入力から英語の文を読んで，間違っている疑いのあるものを出力します

## 使い方

$ git clone ...
$ cd robosys2025
$ echo aho | ./xspell
aho: 間違っている

自分の物に合わせて要編集
